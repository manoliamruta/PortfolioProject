View Portfolio
---------------------------------------------------------------------------------------------------------------
1. Unzip the PortFolioProject.zip. 
2. Open any of the web-browsers. (Mozilla, Chrome etc..)
3. Go to File open, PortFolioProject/index.html.
4. View the PortFolio.
